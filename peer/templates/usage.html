Getting Started

Select a file for analysis and ensure it's in one of the following formats: .docx, .odt, or .txt. Microsoft Word, LibreOffice, and Google Docs can all "export" or "download as" to one of these formats. Be sure of the location of the prepared document on your local storage.

Once you have your document prepared and ready, it's time to upload the document. Click the Select File... button on the upload page. Once you've selected the file, simply click the Submit button beneath the file dialog on the same uploads page.
